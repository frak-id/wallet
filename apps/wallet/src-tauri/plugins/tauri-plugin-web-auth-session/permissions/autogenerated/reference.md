## Default Permission

Default permissions for the web-auth-session plugin

#### This default permission set includes the following:

- `allow-start-web-auth-session`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`web-auth-session:allow-start-web-auth-session`

</td>
<td>

Enables the start_web_auth_session command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`web-auth-session:deny-start-web-auth-session`

</td>
<td>

Denies the start_web_auth_session command without any pre-configured scope.

</td>
</tr>
</table>
