## Default Permission

Default permissions for the frak-firebase plugin (combined FCM + Crashlytics surface).

#### This default permission set includes the following:

- `allow-get-token`
- `allow-request-permissions`
- `allow-check-permissions`
- `allow-register`
- `allow-delete-token`
- `allow-create-channel`
- `allow-send-notification`
- `allow-set-user-id`
- `allow-set-key`
- `allow-log`
- `allow-record-error`
- `allow-set-collection-enabled`
- `allow-test-crash-native`
- `allow-test-rust-panic`
