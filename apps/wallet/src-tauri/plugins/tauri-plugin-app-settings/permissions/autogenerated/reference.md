## Default Permission

Default permissions for the app-settings plugin

#### This default permission set includes the following:

- `allow-open-notification-settings`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`app-settings:allow-open-notification-settings`

</td>
<td>

Enables the open_notification_settings command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`app-settings:deny-open-notification-settings`

</td>
<td>

Denies the open_notification_settings command without any pre-configured scope.

</td>
</tr>
</table>
