# Context7 — Fetch Live Documentation

When a task involves a **library, framework, SDK, CLI tool, or cloud service** (even well-known ones like React, Next.js, Elysia, Drizzle, Viem, SST, Pulumi…), load the `find-docs` skill to fetch current documentation via the `ctx7` CLI.

- **Use for:** API syntax, configuration options, version-specific behavior, setup instructions, library-specific debugging, CLI usage.
- **Do NOT use for:** refactoring, general programming concepts, business logic debugging, or code review.

Training data may be outdated — prefer `find-docs` over assumptions or generic web search.
